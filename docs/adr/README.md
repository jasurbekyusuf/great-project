# Architecture Decision Records

Each numbered file captures one significant decision. They are immutable — when
a decision is reversed, add a new ADR that supersedes the old one rather than
editing history.

Template:

```md
# ADR-NNNN: <Short title>

- **Status**: proposed | accepted | superseded by ADR-MMMM
- **Date**: YYYY-MM-DD

## Context
Why we needed to make a decision.

## Decision
What we decided.

## Consequences
Trade-offs we accept.
```
